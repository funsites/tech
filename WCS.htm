Make more space on Windows computer.

Last edited: 3rd January 2016


DESCRIPTION


Step 1: First, there is a program that comes with your computer. It is called Disk Clean-up. Run it.

Step 2: Once Disk Cleanup finds unnecessary files, allow it to delete them.

Step 3: Get a USB flash drive and put some files on it.

Step 4: Now that some files have been saved onto the USB flash drive, you can delete them from the actual computer.

Step 5: Empty the Recycle Bin on the computer.
